## Golang_Run_ksql

### 需在go環境下執行
- 若沒有Golang環境，請依照[官方](https://golang.org/dl/)安裝、配置
    - Windows:若golang無法在git bash軟體上使用的話，請轉用「cmd」軟體執行專案
- 請確保此專案已放置於
    - Linux/MacOS:`$GOPATH/src/`下
        - $GOPATH 應為  **/Users/[您的使用者名稱]/**
    - Windows:`%GOPATH%/src/`下(若沒有此路徑，請自行新增資料夾)
        - %GOPATH% 應為 **C:**w
